## 貼圖 (Sticker)

### 1. 圖片格式

| # | 規則 |
|---|------|
| 1.1 | 不符合 LINE 指定格式 |
| 1.2 | 不適合日常對話使用 |
| 1.3 | 能見度差（過長、全身像等） |
| 1.4 | 缺乏變化（純色系、數字序列等） |
| 1.5 | 只有 logo |
| 1.6 | **只有文字或短句** ← 圖上文字過多時注意 |
| 1.7 | 含拼字錯誤 |
| 1.8 | 標題與描述內容矛盾 |
| 1.9 | 主圖／tab 圖與實際貼圖完全不同 |
| 1.10 | 與已上架或審核中的貼圖重複 |

### 2. 文字（標題、描述、版權）

| # | 規則 |
|---|------|
| 2.1 | 不符合指定格式 |
| 2.2 | 拼字或語法錯誤 |
| 2.3 | 含廣告性文字（如「11/17 上市」） |
| 2.4 | 含 URL |
| 2.5 | 含 emoji 或跨裝置顯示不一致的字元 |
| 2.6 | **過短的文字** |
| 2.7 | 文字與圖片內容矛盾 |

## 本系列貼圖風險對照

### 情緒嘴硬系列（v1–v3）踩線風險

| 內容 | 規則 | 風險 |
|------|------|------|
| 描述含「嘴超賤」「欠揍」 | 3.11 | 高 |
| 嘴硬情緒如「滾」「吵屁」 | 3.11 | 中 |
| やんのか（要打架？）+ 舉拳圖 | 3.11 / 3.21 | 中-高 |
| あっち行け（滾開）+ 踢人動作 | 3.11 | 中 |
| 「欠揍」「打」等打架語彙 | 3.11 / 3.21 | 高 |

### 安全用詞參考

| 原本 | 替換建議 |
|------|---------|
| 嘴超賤 | 嗆嘴 / 毒舌 |
| 欠揍 | 促咪 / 嗆人 |
| 滾 → 圖片 OK，描述改 | あっちへ |
| やんのか | なんだよ / ちょっと！ |
| あっち行け | あっちへ / もう！ |
